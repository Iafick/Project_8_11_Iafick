## Project Overview

The Student Course Registration System is a Python application designed to manage student information, course registration, and grades.

This project demonstrates concepts from Python Crash Course Chapters 8-11, including:

- Object-oriented programming
- Functions and modularization
- File I/O
- JSON data persistence
- Exception handling
- Automated testing using unittest

## Features

- Register new students
- Add courses
- Enroll students into courses
- Enter and update grades
- Calculate student averages
- Save student information using JSON
- Load saved student information when the program starts
- Automated testing for core functionality
